# Task Management System

ASP.NET Core 8-ზე აგებული სრულფასოვანი RESTful API, რომელიც იყენებს Clean Architecture-ს და მოიცავს CRUD ოპერაციებს მომხმარებლებისთვის, პროექტებისთვის, ამოცანებისთვის და კომენტარებისთვის.

---

## გამოყენებული ტექნოლოგიები

| ფენა | ტექნოლოგია |
|------|------------|
| API | ASP.NET Core 8 Web API |
| ORM | Entity Framework Core 8 |
| მონაცემთა ბაზა | SQL Server (LocalDB) |
| ვალიდაცია | FluentValidation |
| ობიექტების მაპინგი | AutoMapper |
| ტესტირება | xUnit + Moq + FluentAssertions |
| დოკუმენტაცია | Swagger / OpenAPI |
